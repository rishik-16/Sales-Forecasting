# 📈 Sales Forecasting & Demand Intelligence Dashboard

An end-to-end retail sales forecasting and analytics dashboard built with **Python, Machine Learning, and Streamlit**. The project analyzes historical Superstore sales data, forecasts future demand using **XGBoost**, detects anomalies, and segments products to support inventory and supply chain decisions.

## Project Structure

```text
Sales-Forecasting/
├── app.py                 # Streamlit dashboard
├── analysis.ipynb         # Data analysis & model development
├── train.csv              # Superstore sales dataset
├── requirements.txt       # Project dependencies
├── charts/                # Generated visualizations
└── README.md
```
